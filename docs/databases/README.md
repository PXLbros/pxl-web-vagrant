# Databases

## Commands

### `create_database`

#### Interactive

#### Noninteractive

##### Options

* `--driver` (Driver/type, .e.g `--driver=mysql`)
* `--name` (Name of database, .e.g. `--name=my_database`)

```shell
create_database \
    --driver=mysql \
    --name=my_database
```

### `delete_database`

#### Noninteractive

##### Options

* `--driver` (Driver/type, .e.g `--driver=mysql`)
* `--name` (Name of database, .e.g. `--name=my_database`)

```shell
delete_database \
    --driver=mysql \
    --name=my_database
```
